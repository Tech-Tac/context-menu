# Context Menus

A custom context menu library for web projects!

Showcase: <https://tech-tac.github.io/context-menu/>

## Features

- Extendability
- Customizability
- Ease of use
- Sub-menus
- Icons
- Headers
- Separators
- Disabled items
- Click actions
- Keyboard accessable
- Touch screen support
- And more to come!
